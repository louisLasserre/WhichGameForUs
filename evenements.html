Évènements

Which Game For Us présente “Tous Pour Le Jeu”, une série de conférences diffusées live sur Twitch. Les thèmes abordés seront autour de l’accessibilité dans les jeux-vidéo, avec pour objectif de rassembler les grands acteurs de l’industrie du jeu vidéo et les joueurs concernés. Nous animons des discussions concernant les pratiques à mettre en place pour une meilleure inclusion dans les jeux vidéo.
